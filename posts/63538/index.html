Line Notify 處理服務綁定的套件

相關連結

  • NPM
  • Github Repository

關於此作品

這個作品主要是當時在撰寫 水藍嗶啵鼠 時要接 Line Notify 的過程中有很多設定與文件要看，後來乾脆索性自己開發一個套件專門解決這件事情，就像用 Node.js 中的 Passport 一樣；主要是解決將使用者與服務綁定的中間這段 Oauth 2 的認證過程。

  • 2020/7 開發完成
  • 主要使用了 express 中的路由功能實作，使用者只須引入套件後填入以下欄位即可串接服務
    1. client_id
    2. client_secret,
    3. succeeded_route,
    4. failed_route,
