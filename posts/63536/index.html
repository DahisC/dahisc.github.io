The F2E 2nd 第十關 - 口罩地圖

Author: DahisC Date: July 17, 2021  19:47:36 Category: Portfolio

相關連結

  • Github Repository
  • 作品網址 (Heroku)

關於此作品

六角學院的 The F2E 2nd 活動，使用了 2020 年初政府因應疫情擴散推出的 健保特約機構口罩剩餘數量明細清單 API 實作以地圖為服務的口罩地圖。

  • 2020/4 開發完成
  • 以 Nuxt 開發，並以 Vuex 管理狀態

使用技術

技術／套件
前端框架 Vue/Vuex
後端框架 Nuxt
地圖套件 Leaflet
部署環境 Heroku
第三方 API 健保特約機構口罩剩餘數量明細清單 API
