兩分鐘將部落格系統 Hexo 部署至 Github Pages

Author: DahisC Date: June 27, 2021  14:29:32

前言

什麼是 Github Pages？

Github Pages 是 Github 提供的免費靜態網頁部署空間，每個帳號都可以免費獲得一個對應 Github 帳號的空間。

什麼是 Hexo？

Hexo 是一個快速、簡單且強大的網誌框架。Hexo 使用 Markdown（或其他標記語言）解析您的文章，並在幾秒鐘內，透過漂亮的主題產生靜態檔案。

這文章要幹嘛？

用兩分鐘教你怎麼在 Github 上建立一個部落格系統！！！

前置作業

下載並安裝

  1. Node.js
  2. git

套件

  1. npm i hexo-cli -g

步驟

以下的步驟都以 Github 帳號 Hexo 為例

1. 在 Github 上新增一個公開儲藏庫

2. Clone 儲藏庫至本機並在根目錄中執行以下指令

3. 更改根目錄中 _config.yml 的部署設定

4. 在 /blog 底下執行以下指令

5. 更改儲藏庫設定

  • 設定 Github Pages 的檔案來源

    1. 將 Branch 設定成 gh-pages

6. 完成

  • 部署的網站會出現在 https://Hexo.github.io/
